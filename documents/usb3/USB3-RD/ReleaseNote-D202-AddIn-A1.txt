1. Summary
---------------------------------------------------------------------------
 This release note explaina Design Kit of ET-D720202-0014 for USB 3.0
 Host Controller uPD720202.
 This design supports USB wake-up function. 
 If you do not need to support this wake-up function, please refer the 
 schematics including this Design Kit to remove some circuits.
